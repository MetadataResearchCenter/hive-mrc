# Release Notes #

The HIVE 2.2 release includes the following major features:

  * Ability to index a specific locale in a multilingual thesaurus
  * Add NCBI, GeoChronoStat, FAST, LCSH Spanish
  * Fix TGN issue
  * Load latest AGROVOC
  * Fix bug mentioned by UC3M
